🌟 Overview

This project introduces a cutting-edge AI-powered proctoring system designed to maintain fairness, security, and integrity in remote assessments. By leveraging advanced machine learning techniques, this system redefines how online tests are monitored.

Troubleshooting Common Issues

If you encounter issues while setting up the project, ensure the following:

  • All dependencies are installed using pip install -r requirements.txt.
  • Ensure the correct Python version (3.8 or above) is installed.
  • Check database connectivity for PostgreSQL.
  • Review the environment variables setup as per the documentation.
